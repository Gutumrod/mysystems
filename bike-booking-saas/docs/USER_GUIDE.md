# คู่มือใช้งาน Bike Booking SaaS

## สำหรับลูกค้า

1. เข้าเว็บของร้าน
2. กรอกชื่อ เบอร์โทร Facebook หรือ LINE ID ถ้ามี
3. เลือกรุ่นรถ ปีรถ วันที่ เวลา และบริการอย่างน้อย 1 รายการ
4. กด `ยืนยันการจอง`
5. เมื่อจองสำเร็จ ให้กดคัดลอกข้อความแล้วส่งให้ร้านผ่าน LINE/Facebook ได้ทันที

## สำหรับร้าน

### Dashboard

- ดูจำนวนคิววันนี้ สัปดาห์นี้ เดือนนี้
- ดูรายการคิววันนี้เรียงตามเวลา
- เปลี่ยนสถานะเป็น เริ่มทำ, ทำเสร็จ, ยกเลิก หรือ No-show

### ปฏิทิน

- ดูคิวแบบเดือน สัปดาห์ วัน
- สีแสดงสถานะของคิว
- คลิกคิวเพื่อดูรายละเอียด

### การจอง

- ค้นหาด้วยชื่อ เบอร์ หรือรุ่นรถ
- กรองตามวันที่และสถานะ
- เปิดรายละเอียดเพื่อแก้ไขสถานะหรือยกเลิก

### บริการ

- เพิ่ม แก้ไข ซ่อน และลบบริการ
- ลากเรียงลำดับบริการ
- ถ้าบริการถูกใช้ใน booking แล้ว ระบบจะป้องกันการลบ

### ตั้งค่า

- แก้ไขข้อมูลร้าน เบอร์ LINE และ Facebook
- ตั้งเวลาเปิดปิด 7 วัน
- เพิ่มวันหยุดพิเศษพร้อมเหตุผล

## กฎสำคัญ

- ลูกค้าจองแล้วสถานะเป็นยืนยันทันที
- ระบบไม่แสดงราคา เพราะร้านประเมินหลังตรวจหน้างาน
- ถ้าลูกค้า No-show 3 ครั้ง ระบบจะ blacklist และแจ้งให้ติดต่อร้านโดยตรง
